GET /api/v1/payments

### Query Parameters

| Parameter | Type | Default | Description |
| :-- | :-- | :-- | :-- |
| page | integer | 1 | Page number |
| pageSize | integer | 20 | Number of results per page |
| sortBy | string | created\_at | Sort field (created\_at or status) |
| sortOrder | string | desc | Sort order (asc or desc) |
| status | string | - | Filter by status (see [Types](./types)) |

### Example Request

curl "[https://api.chainpal.org/api/v1/payments?page=1&pageSize=10&status=completed](https://api.chainpal.org/api/v1/payments?page=1&pageSize=10&status=completed)" \
-H "Authorization: Bearer cp\_sk\_test\_xyz789"
