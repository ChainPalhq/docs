#  Types & Enums

Reference for common types and enum values used in the API.

## Order Status

The status of a payment.

| Value        | Description                                    |
| :----------- | :--------------------------------------------- |
| `pending`    | Payment initialized but no funds detected yet. |
| `processing` | Funds detected, payment is processing.         |
| `completed`  | Payment confirmed and funds settled.           |
| `failed`     | Payment failed (e.g., underpayment, timeout).  |
| `expired`    | The payment window has expired.                |

## Settlement Currencies

Supported fiat currencies for **settlement** (payout to bank accounts).

- `NGN` - Nigerian Naira
- `GHS` - Ghanaian Cedi
- `KES` - Kenyan Shilling
- `ZAR` - South African Rand

> **Note:** `USD` is supported for **collection pricing** (e.g., charging \$100), but settlement will occur in the local currency of the merchant's country. We do not currently support direct USD settlements.

## Transaction Limits

The following limits apply to payment amounts based on the merchant's country.

### Minimum Fiat Amounts

- **Nigeria (NGN)**: 1,000.00
- **Ghana (GHS)**: 20.00
- **Kenya (KES)**: 50.00
- **South Africa (ZAR)**: 35.00

### Maximum Fiat Amounts

- **Nigeria (NGN)**: 1,000,000.00
- **Ghana (GHS)**: 5,000.00
- **Kenya (KES)**: 250,000.00
- **South Africa (ZAR)**: 50,000.00

### USD Collection Limits

- **Minimum**: \$1.00 (Global)
- **Maximum**: \$5,000.00 (Global)
